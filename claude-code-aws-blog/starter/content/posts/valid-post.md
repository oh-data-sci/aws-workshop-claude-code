# Getting Started with AWS Bedrock

AWS Bedrock provides a powerful platform for building production-ready AI
applications. In this guide, we'll explore how to integrate Claude Code with AWS
Bedrock to streamline your development workflow.

## Key Features

- **Scalable AI Infrastructure**: Leverage AWS's proven cloud infrastructure
- **Multiple AI Models**: Access to various foundation models including Claude
- **Enterprise Security**: Built-in security and compliance features
- **Cost Optimization**: Pay only for what you use

## Getting Started

To begin working with AWS Bedrock, you'll need:

1. An AWS account with Bedrock access enabled
2. Proper IAM permissions configured
3. The AWS CLI installed and configured
4. Claude Code installed and ready to use
